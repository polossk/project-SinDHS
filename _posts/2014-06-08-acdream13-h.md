首先是要姿势正确！

注意完美数的生成机：2+2a+2b+ab（ab都是完美数）

假设生成完美数c

c = 2 + 2a + 2b + ab

c + 2 = ab+2a+2b+4

c + 2 = (a + 2)(b + 2)

然后一开始只有两个完美数1和3.

所以所有的完美数只有质因数分解之后都是类似于

2 + N = (3 ^ x) * (5 ^ y)

但是5不是完美数。然后就没事了。。。
